# VCS Dashboard - Frontend

This is the frontend dashboard for the Custom Version Control System (VCS). It allows users to visualize repositories, commit histories, and file contents in a clean, GitHub-inspired interface.

## 🚀 Features

- **Repository Browser**: View all registered repositories.
- **Commit History**: See a detailed list of commits, including messages and timestamps.
- **File Preview**: Explore the files included in each commit.
- **Contribution Graph**: Track repository activity over time (Integrated with Backend).
- **Responsive Design**: Modern UI built with React and Vanilla CSS.

## 🛠️ Tech Stack

- **React**: Library for building the user interface.
- **Vite**: Next-generation frontend tooling for fast development.
- **CSS3**: Custom styles for a premium, dark-themed experience.
- **Lucide React**: For beautiful icons.
- **React Router**: For seamless navigation.

## ⚙️ Setup Instructions

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/sawantyash07/AWS-frontend.git
    cd AWS-frontend
    ```

2.  **Install dependencies**:
    ```bash
    npm install
    ```

3.  **Configure Backend**:
    Ensure the [Backend](https://github.com/sawantyash07/VCS-Backend) (link placeholder) is running on `http://localhost:5000`.

4.  **Start the development server**:
    ```bash
    npm run dev
    ```

5.  **Build for production**:
    ```bash
    npm run build
    ```
